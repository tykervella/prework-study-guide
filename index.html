HTML

HTML is used to create the structure of a website. It is made of elements which work together to provide content and the basic framwork for all webpages

General Notes

  • The head element contains information about the webpage, called metadata
  • The body element represents the visible content shown to the user.
  • Elements work together to build the structure of a webpage and provide a framework that makes it easy to navigate


Examples of Some Basic Elements and What They Do:

All below elements are contain within brackets "<" ">"
  • !DOCTYPE html this is a declaration to tell the browser what document type to expect
  • html lang="en" this is a opening tag that represents the root of our html doc
    • Inside the opening tag is the attricute for the langauge of the webpage
    • Attributes define additional info about an element and are always defined in the beginning element
  • head an opening element that contains metadata tags
    • meta defines metadata such as the character set, description, keywords, author, and viewport
    • meta charset defines the charset... UTF-8 is one set which allows the use of most characters and symbols on a page
    • meta http-equiv sets an HTTP header for the content
    • meta viewport sets the default view for a webpage
  • title defines the title for the website and is important for appearing in search enginges
  • style defines the CSS styles
  • base defines the base URL
  • link connects an external resource
  • script embeds client-side scripts
  • h1 indicates header text
    • There are various levels of headers that go from h1 to h6
  • p indicates a paragraph or block of text. used to wrap texts
  • ul, ol, and li indicates unordered lists, ordered lists and list items respectively
  • img contains information about images that're contained on a webpage
    • this can contain elements that have various attributes, such as "src" which defines the location of the image
  • br creates a line break between two blocks of content
  • a known as an anchor, creates a link to the same webpage or another
    • the href attribute points to the url for the link
    • alt attribute contains a text string that describes the apperance and functionality of an image and allows the use of assisstive technologies.
  • section indicates a semantic element that defines a section in a document
    • the id attribute indentifies elements and allows them to be selected
    • class attribute COMING SOON


CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status tells us which branch we're currently on
  • git checkout -b feature/text creatures a new branch titled feature/text where we can save our WIP
    • checkout command moves the working branch to a new branch
    • -b flag creates a new branch
    • feature/text is the name of the new branch. the prefix "feature/" indicates that this is to work on a specific feature with this branch
  • git add -A adds modifications in the current working branch to the staging area
    • -A tag indicates that we want to add all changes
    • the staging area inform Git that these changes will be added to the next commit
  • git commit -m "text" commits what is in the staging area, with text explaining what the updated code has done
  • git pull origin main
    • pull command receives branch modifications into our local environment
    • origin keyword indicates the source of the pull will be in the github repository
    • main keyword indicates the branch we are pulling from
  • git push origin text pushes our changes to the remote branch text
